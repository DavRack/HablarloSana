# AI Agent Instructions for HablarloSana

## Project Overview

You are working on **HablarloSana**, a customer-facing web application for a psychology practice.

## Core Directives

1. **Security First**: Patient data and privacy are paramount. Never write code that exposes sensitive information or uses insecure data handling practices.
2. **Maintainability**: Write clean, modular, and extensively documented code.
3. **Premium Aesthetics**: The UI must be modern, responsive, and visually stunning. Use curated color palettes, micro-animations, and dynamic designs to create a calming, premium experience for patients. Avoid generic or basic UI layouts.
4. **Tone**: Code comments and commit messages should be professional and clear.

## Technology Stack

- **Frontend Framework**: SvelteKit (Static Adapter)
- **Deployment**: GitHub Pages (Custom Domain via GitHub Actions)
- **Styling**: Vanilla CSS with modern variables.

## Design System

**Color Palette:**

- **Cool Steel**: `#91A2BA`

## Workflow

- When asked to create new features, ensure they align with the secure and maintainable architecture.
- Always validate that new dependencies are secure and necessary before adding them.
- Before making significant architectural changes, propose them to the user for confirmation.
